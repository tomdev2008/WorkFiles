1、修改添加虚拟机的powershell脚本。
$VHD = Get-SCVirtualHardDisk -VMMServer "syscen102.syscen95.com" | where { $_.Name -eq "08-clean.vhd" -and $_.LibraryServer -eq "syscen60.syscen95.com"}

$VMHost = Get-SCVMHost | where {$_.Name -match "syscen60.syscen95.com"}

$HWProfile = Get-SCHardwareProfile | where {$_.Name -match "hardwarepf"}

New-SCVirtualMachine -Name "VM1" -Description "New Virtual Machine by vmm script" -MemoryMB 4096 CCPUCount 2 -VirtualHardDisk $VHD -VMHost $VMHost -Path "D:\Hyper-V\" -HardwareProfile $HWProfile
当$VHD,$VMHost,$HWProfile三个参数不为空的时候才能创建虚拟机，否则提示参数错误，创建失败。
